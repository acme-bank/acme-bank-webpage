ACME Bank

ACME Bank is a fictitious bank used as the business case for developing a business process platform using modern lightweight enterprise architecture.

Purpose

The purpose of this project is to have sensible business case to use as the backdrop for developing modern solutions using the most cutting edge technologies and the most modern architecture in the market.

As well as the software itself, all of which is open source, this project tries to show how to develop and deploy the software using modern runtimes and methodologies.
