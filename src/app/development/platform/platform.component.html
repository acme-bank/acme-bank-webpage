DevOps Platform

The ACME Bank DevOps platform is a runtime and CI/CD enabling platform for development of ACME applications.

Platform services

The platform is a container based framework based on Docker, and is comprised of the following services.

  • Docker Registry - Registry to version ACME Docker images
    • Docker Registry UI - User interface for the Docker Registry
  • Jenkins - CI build server
  • Nexus - Artifact repository
  • Apache Kafka - Mesage broker and event store
    • Apache ZooKeeper - Distributed service orchestration
  • HAProxy - Proxy server and load balancer
